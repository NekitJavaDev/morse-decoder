#### Решаемая научно-прикладная задача:
Разработка ПО, обрабатывающее звуковые файлы, содержащее азбуку Морзе.

#### Алгоритм программы:
1. Загрузить файл из ОС ( .mp3 или .wav) или записать файл через микрофон
2. Распарсить выбранный файл в символы азбуки Морзе (отфильтровать шумы, сформировать спектограмму и т.п.)
3. Перевести из сиволов Морзе на выбранной при старте программы язык

#### Поддерживаемые ПО языки (перевод символов):
* Russian (RU)
* English (ENG)

#### Недостатки ПО:
* Поддерживает только моно звуковые файлы (при загрузке стерео файла считывается только 1 канал) - доделать
